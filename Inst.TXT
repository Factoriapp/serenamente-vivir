Creación de Clases SCSS (Devito)
Las clases de layout se añadirán a _sections.scss y las clases de formulario a _buttons.scss o, si existe un _forms.scss, ahí. Asumiremos que las clases de formulario se pueden añadir a _sections.scss para simplificar si _forms.scss no existe.

Instrucción para Devito (Claude Code):

Devito, implementa las siguientes clases en scss/_sections.scss y compila.

Clases de Contenedor y Estructura:

SCSS

/* Contacto: Estructura de dos columnas con espaciado */
.contact-main-content {
    padding: var(--spacing-xl) var(--spacing-md); /* Mapeo de 80px 40px */
    margin-bottom: var(--spacing-lg); /* 60px */
}
.contact-grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: var(--spacing-md); /* 40px */
}
/* Cajas de Formulario */
.form-box-contact {
    padding: 50px;
    border-radius: 15px; /* Usar valor explícito por si no hay variable exacto */
    box-shadow: var(--shadow-soft);
    background-color: white;
}
.form-box-appointment {
    @extend .form-box-contact; 
    background: linear-gradient(to bottom, var(--color-green), white);
    border: 3px solid var(--color-beige);
}
/* Contenedor de Información de Contacto */
.contact-info-block {
    background-color: var(--color-cream);
    padding: var(--spacing-md); /* 40px */
    border-radius: 15px;
}
.contact-info-item {
    display: flex;
    align-items: center;
    gap: 15px;
    margin-bottom: 15px;
    color: #666;
    strong {
        color: var(--color-gray);
        min-width: 80px;
    }
    a, span {
        color: var(--color-gray);
        text-decoration: none;
    }
}
```
Clases de Formulario (Inputs, Select, Textarea):

SCSS

.form-field-group {
    display: flex;
    flex-direction: column;
    gap: 8px;
}
.form-label {
    font-size: 14px;
    font-weight: 600;
    color: var(--color-gray);
}
.form-input-style {
    padding: 12px 16px;
    border: 2px solid var(--color-beige);
    border-radius: 8px;
    font-size: 16px;
    background-color: white; /* Para consistencia, especialmente en el bloque de cita */
    &:focus {
        border-color: var(--color-cyan);
        outline: none;
    }
    &.textarea {
        resize: vertical;
        min-height: 120px;
    }
}
Compila: npm run build.
